Phase 2: Feature Engineering & Model Design

Duration: 2-3 Weeks

In this phase, we translate raw code into a format our model can understand. We will define the features for training our review model by parsing the code's structure and then design a suitable machine learning architecture to learn from these features.

  • ‚úî
    AST Parsing: Develop a parser to convert Python code into Abstract Syntax Trees (ASTs).

    Feature Extraction: Extract features from ASTs relevant to ML best practices (e.g., `train_test_split` usage).
  • ‚úî
    Model Selection: Choose a model architecture, likely a Graph Neural Network (GNN), with a simpler baseline.

    Labeling: Annotate the code dataset with identified issues to create labels for supervised learning.
